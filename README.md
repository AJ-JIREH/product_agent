# product_agent
Description
This project implements an AI-powered product chatbot using Llama Index and OpenAI.
The chatbot can:
•	Answer questions like:
o	“What products are available?”
o	“Which products are under 50?”
o	“Tell me about product X”
•	Automatically calls a product search tool to retrieve approved products.
•	Supports multi-turn conversations using a memory buffer.
•	Stores chat history ).
Why Llama Index?
It allows the agent to:
•	Reason over structured data (product list)
•	Decide when to call tools
•	Support memory for multi-turn conversations
•	Easily integrate with LLMs like OpenAI
